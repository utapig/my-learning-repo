## AI&app-development

## 基本方針
- 画面を綺麗に作成できるとモチベーションが上がるので、小さいアプリでも必ず画面まで作成する
- 通勤中もアウトプットを含めた学習をしたいので、クイズを出してもらいながら学習する
- コードは基本的にAIではなく自分で書き、理解することを重視する

## 学習内容

| 技術 | 理由 |
| :--- | :--- |
| java | 業務で使うから。また、プログラミング言語の基礎を学べるから。 |
| React | フロントに興味があるから。 |

## 学習方法

| 場面 | 内容 | ツール名 | 理由 |
| :--- | :--- | :--- | :--- |
| 通勤時間 | AIにメンターとなってもらい、クイズを出してもらいながら体系的に学習する。 | ChatGPT | 使い慣れているため |
| 休日 | ミニアプリをAIにサポートしてもらいながら開発する。UIデザインにはこだわる。 | github copilot | 多くのユーザーが使っているため & 業務でも使っているため |
